# Đồ án Cơ sở Ngành: Xây dựng mô hình giao diện web dựa trên nghiên cứu UX/UI

## Thông tin đồ án
- **Tên đồ án**: Xây dựng mô hình giao diện web dựa trên nghiên cứu UX/UI
- **Đơn vị**: Khoa Công Nghệ Thông Tin - Trường Đại học Trà Vinh
- **Công nghệ sử dụng**: HTML5, CSS3, JavaScript, Bootstrap 5

## Mô tả dự án
Website giới thiệu Khoa Công Nghệ Thông Tin với giao diện hiện đại, thân thiện với người dùng, được thiết kế dựa trên các nguyên tắc UX/UI.

## Tính năng chính
- ✅ Giao diện responsive, tương thích mọi thiết bị
- ✅ Hiệu ứng animation mượt mà
- ✅ Navigation menu với smooth scrolling
- ✅ Sections: Trang chủ, Giới thiệu, Chương trình đào tạo, Tin tức, Liên hệ
- ✅ Form liên hệ tương tác
- ✅ Scroll to top button
- ✅ Thiết kế theo chuẩn UX/UI hiện đại

## Cấu trúc thư mục
```
project/
```

## Hướng dẫn sử dụng
1. Mở file `TRANGCHU.html` bằng trình duyệt web
2. Hoặc sử dụng Live Server trong VS Code để xem với hot reload

## Các thư viện sử dụng
- Bootstrap 5.3.0 - Framework CSS
- Font Awesome 6.4.0 - Icon library
- Google Fonts - Typography

## Nguyên tắc UX/UI được áp dụng
1. **Hierarchy**: Sử dụng kích thước, màu sắc để tạo thứ bậc thông tin
2. **Consistency**: Nhất quán về màu sắc, font chữ, spacing
3. **Responsive Design**: Tương thích mọi kích thước màn hình
4. **Visual Feedback**: Hiệu ứng hover, transition cho các tương tác
5. **Accessibility**: Dễ dàng điều hướng, đọc hiểu
6. **White Space**: Sử dụng khoảng trắng hợp lý
7. **Color Psychology**: Màu xanh dương thể hiện sự chuyên nghiệp, tin cậy
